# `ui-button-group`

#### `can be instanciated`

```html
<ui-button-group>
</ui-button-group>

```

#### `can be created`

```html
<ui-button-group>
</ui-button-group>

```

#### `has a default slot`

```html
<slot>
</slot>

```

#### `can host ui-button`

```html
<ui-button-group>
  <ui-button role="button">
    button
  </ui-button>
</ui-button-group>

```

#### ``has `buttons` property``

```html
<ui-button-group>
  <ui-button role="button">
    a
  </ui-button>
  <ui-button role="button">
    b
  </ui-button>
</ui-button-group>

```
